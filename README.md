# TalkToYou - Real-time Chat Application

실시간 채팅 애플리케이션을 통한 DevOps 파이프라인 구축 프로젝트

## 🛠 기술 스택
- Frontend: React.js
- Backend: Spring Boot
- Database: MongoDB, Redis
- Infrastructure: AWS EKS, Terraform, Docker, Kubernetes
- CI/CD: Jenkins, ArgoCD
- Monitoring: Prometheus, Grafana
